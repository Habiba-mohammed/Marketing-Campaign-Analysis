#  Marketing Campaigns Analysis Dashboard

![Marketing Campaign Dashboard](Marketing_Campaign.png)

##  Project Overview
This project presents an interactive **Marketing Campaigns Analysis Dashboard** built using **Power BI**. The goal is to transform raw marketing data into meaningful insights that help stakeholders evaluate campaign performance and make data-driven decisions.

---

## Step 1: Data Cleaning (Power Query)
The first and most critical step in this project was **data cleaning using Power Query**.

### Key actions:
- Removed duplicates and handled missing values  
- Standardized column names and formats  
- Fixed data types (dates, numbers, categories)  
- Removed invalid or inconsistent records  

This step ensures **clean, accurate, and reliable data** before analysis.

---

## Step 2: Data Modeling (High Performance)
After cleaning, a **data model** was built for performance and scalability.

### Key features:
- Star schema design (Fact & Dimension tables)  
- Relationships between tables  
- Optimized filtering and performance  

This step ensures **fast and efficient analysis** in Power BI.

---

## Step 3: Key Performance Indicators (KPIs)

The most important KPIs were created to evaluate marketing campaigns clearly:

### Core KPIs:
- **Total Sales** → Total revenue generated  
- **Conversion Rate** → % of users converted  
- **Campaign ROI** → Profit vs cost of campaigns  
- **Customer Acquisition** → Number of new customers  
- **Engagement Metrics** → Interaction with campaigns  

These KPIs help answer:
- Which campaigns are successful?  
- Where should budget be increased?  
- What needs improvement?  

---

## Insights & Value

- Supports data-driven decision making
